# NetHub Documentation

## Overview
NetHub is an enterprise-grade home network management suite.

## Configuration
To customize your network and devices, edit the following files:
- `config/network.json`: Define your network nodes (Routers, NAS, Printers, etc.) and IoT devices.

## Folder Structure
The project is organized into 9 main root folders:
1. `config`: JSON configuration files for devices and network settings.
2. `src`: React application source code.
3. `public`: Static assets like icons and manifest files.
4. `media`: Local storage for the Media Server.
5. `docs`: Project documentation and guides.
6. `scripts`: Automation and utility scripts.
7. `tests`: Unit and integration tests.
8. `dist`: Production build output.
9. `node_modules`: Project dependencies.
